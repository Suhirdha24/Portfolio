Projects

Multiple Disease Prediction using Machine Learning

Developed a machine learning model using Random Forest and SVM to predict multiple diseases, including diabetes, heart disease, and cancer. The project involved collecting and cleaning healthcare data, training the model, and evaluating its performance using accuracy, precision, and recall metrics. I also visualized the results using Python and Scikit-learn, helping healthcare professionals with early diagnosis decisions.

Wildfire Prediction using Deep Learning

Built a Convolutional Neural Network (CNN) model using TensorFlow and Keras to predict the presence of wildfire in images taken from remote areas. The system analyzes real-time satellite and drone imagery to detect early signs of fires. The project involved data preprocessing, model training, and deployment, which could potentially save lives by providing real-time alerts to firefighters and authorities.

Smart Parking System using YOLO

Implemented a real-time parking detection system using the YOLOv8 algorithm, designed to automatically detect available parking spaces in urban areas by analyzing live video feeds from surveillance cameras. The project integrates computer vision techniques with real-time data processing to ease the burden of finding parking in crowded cities. The system is able to predict open spots with high accuracy and provide dynamic availability updates to users.
